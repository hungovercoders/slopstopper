🔐 About the comments embed

Strict CSP — with one documented exception.

This site ships a strict default-src 'self' Content Security Policy on every path. This page is the only exception: /feedback.html allows https://giscus.app via per-path [[headers]] in netlify.toml, with the loader script SRI-pinned. The exception is documented in docs/security/CSP_EXCEPTIONS.md, and a hygiene check (task ss:hygiene:csp-exceptions) fails the build if either side drifts.

If you're adopting SlopStopper and need to embed a third-party widget (GTM, Sentry, Intercom, an analytics tag) — this is the pattern. Read the full guide →
