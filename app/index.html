SlopStopper is a template and reference site demonstrating how to add deterministic, automated quality feedback to any repo. As AI-assisted development accelerates the pace of change, consistent automated checks keep code quality high without slowing teams down.

Every check you see on Features runs on this repo too — pipeline-status badges on the README are real.

Get started

Install the CLI. Run checks anywhere.

SlopStopper is a Python CLI plus the GitHub Actions workflows that drive it. Install the CLI alone for local use, or run the all-in-one installer to also seed the workflows + Taskfile shim into a repo.

Just the CLI

pipx install git+https://github.com/hungovercoders/slopstopper.git@main#subdirectory=cli
slopstopper checks list
slopstopper doctor
slopstopper run hygiene:docs-size

Installing from git while we get the PyPI release pipeline up; the short pipx install slopstopper-cli form lands soon. pipx upgrade slopstopper-cli already keeps it fresh either way.

Full suite into a repo

curl -fsSL https://raw.githubusercontent.com/hungovercoders/slopstopper/main/install.sh | bash
What does this do?

The installer pip-installs slopstopper-cli (pipx-preferred), then seeds your repo with the .github/workflows/ss-*.yml CI checks, a thin Taskfile.ss.yml of task ss:* shims that call the CLI, and a .slopstopper.yml starter config. Idempotent — re-run any time to refresh.

Bundled Playwright specs, lighthouserc, and the local-CI static server (slopstopper serve) live inside the wheel. Run slopstopper templates eject <name> to customise any of them.

Claude Code users

Let the agent drive the install.

A trio of companion skills — slopstopper-install, slopstopper-update, slopstopper-triage — drops into your Claude Code profile so any project can ask the agent to add, refresh or triage SlopStopper. Claude Code auto-picks the right skill per prompt.

curl -fsSL https://raw.githubusercontent.com/hungovercoders/slopstopper/main/install-skill.sh | bash
What does this do?

The script writes three files — one per skill — under ~/.claude/skills/slopstopper-*/SKILL.md, which Claude Code auto-discovers across every project on your machine. Different prompts trigger different skills: "install slopstopper" → slopstopper-install; "refresh slopstopper" → slopstopper-update; "fix this failing slopstopper check" → slopstopper-triage. Re-run any time to refresh all three to the latest version.
